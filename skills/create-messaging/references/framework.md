# Website Messaging Framework

Always include:

- Sitewide foundations
- Home page copy
- Product or how-it-works page
- Solutions or use cases
- Industries if relevant
- Features catalog
- Pricing
- Customers and proof
- Case studies
- Security and compliance if relevant
- About
- Resources
- FAQ
- Footer

## Sitewide foundations

Include:

- Core positioning statement
- Brand personality traits
- Brand voice and tone rules
- Primary and supporting messages
- Elevator pitch
- Segment-level messaging matrix
- Keyword and intent map
- Trust signals

## Guardrails

- No opening commentary.
- No repeated title header.
- Keep the output conversion-oriented and reusable.
