# Create Ecosystem Messaging

## Overview

Create messaging for integrations and partner ecosystems that emphasizes mutual value rather than one-sided product promotion. Use this for directory listings, joint campaigns, partner pitches, integration launch copy, and alliance sales support.

Read [references/framework.md](references/framework.md) for the joint-value structure and asset list.

## Workflow

1. Confirm the partner or integration type, joint audience, and enabled use cases.
2. Define the combined value proposition and integration story arc.
3. Build use-case messaging and audience-specific variants.
4. Add proof, competitive context, and channel assets.
5. Finish with launch and enablement outputs.

## Output Contract

- Start directly with the framework output.
- Keep the messaging balanced across both sides of the partnership.
- Make the setup, workflow, and joint outcome concrete.
- Include easy-to-adapt templates where the framework calls for them.

## Quality Bar

- Mutual value first.
- Highlight operational simplicity where relevant.
- Keep partner-facing and customer-facing language distinct.
