# Product Insights Framework

Always include:

1. Product-market-fit assessment
2. Voice of customer synthesis
3. Usage pattern insights
4. Competitive feature analysis
5. Feature prioritization matrix
6. Value proposition validation
7. Segmentation insights
8. Messaging opportunities
9. Roadmap recommendations
10. Metrics to track

## Guardrails

- No opening commentary.
- Make recommendations specific and ranked.
- Use a visible logic for prioritization.
- Tie messaging updates to customer evidence.
