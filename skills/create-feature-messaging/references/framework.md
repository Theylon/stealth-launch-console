# Feature Messaging Framework

Always include:

1. Feature positioning statement
2. Feature-benefit ladder
3. Headline options
4. Short-form descriptions
5. Use case messaging
6. Objection responses
7. Competitive positioning
8. Social proof integration
9. Channel-specific messaging
10. Launch messaging sequence

## Guardrails

- No opening commentary.
- Benefits before features.
- Keep messaging tailored to the named audience.
- Use proof and comparisons carefully.
