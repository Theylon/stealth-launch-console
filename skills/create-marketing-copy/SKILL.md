# Create Marketing Copy

## Overview

Create a broad bank of conversion-focused copy assets across formats and persuasion frameworks. Use this when the user needs a multi-format copy system rather than one page or one channel.

Read [references/framework.md](references/framework.md) for the format set and persuasion structures.

## Workflow

1. Confirm the objective, audience, stage, tone, and desired response.
2. Build the copy strategy foundation first.
3. Generate headline banks and framework-based copy blocks.
4. Produce the requested channel formats across web, email, ads, social, and sales.
5. End with conversion and language guidance.

## Output Contract

- Start directly with the marketing-copy framework.
- Generate multiple variants where the framework expects them.
- Keep copy benefit-led and action-oriented.
- Adapt the tone to the audience and funnel stage.

## Quality Bar

- Avoid generic copy that could fit any product.
- Variations should be materially different, not superficial rewrites.
- Persuasion frameworks should serve the goal, not become formulaic clutter.
