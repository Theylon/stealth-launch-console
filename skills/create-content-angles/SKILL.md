# Create Content Angles

## Overview

Find fresh perspectives for a topic instead of repeating the same content market conventions. Use this when the user needs differentiated editorial angles that can be turned into briefs, headlines, and distribution plans.

Read [references/framework.md](references/framework.md) for the angle categories and deep-dive structure.

## Workflow

1. Assess the topic, audience, and what the market has already covered.
2. Generate angle options across the required categories.
3. Score novelty, credibility, engagement potential, and difficulty.
4. Expand the strongest angles into briefs with headlines, hooks, proof needs, and format guidance.
5. Show how the same topic shifts for different audience segments and channels.

## Output Contract

- Start directly with the content-angle framework.
- Make evidence needs explicit for every serious angle.
- Include headline and hook ideas for the top recommendations.
- End with a brief-ready output for the winning angle.

## Quality Bar

- The angles should actually differ from common topic coverage.
- Novelty should not come at the expense of credibility.
- Choose frames that fit the audience's real concerns.
