# Content Angles Framework

Always include:

1. Topic analysis
2. Angle categories
3. Differentiation matrix
4. Deep dive on top 5 angles
5. Audience segment angles
6. Format-angle combinations
7. Cross-platform angle adaptation
8. Angle testing framework
9. Angle refresh strategy
10. Content brief template

## Guardrails

- No opening commentary.
- Prioritize differentiated, supportable perspectives.
- Specify the evidence each angle would need.
- Adapt the strongest angles to real channels and formats.
