# ICP Selection Frameworks

Use these ideas to choose a sharp beachhead instead of a vague target audience.

## Crossing the Chasm

- Start with a beachhead segment, not the whole market.
- Prefer customers with an urgent problem and a clear reason to switch now.
- A good first segment is reachable, referenceable, and similar enough that wins can compound.

## Obviously Awesome

- Look for the best-fit customer, not the largest possible customer.
- The right segment is where your differentiated value matters most against real alternatives.
- If the segment does not care about the difference, it should not drive positioning.

## Practical scoring factors

- Pain severity
- Trigger frequency
- Budget ownership
- Sales friction
- Existing proof
- Reachability
- Expansion value

Use these factors qualitatively unless the user provides actual pipeline or revenue data.
