# Positioning Frameworks

Use these models to create product positioning that downstream copy can inherit.

## Obviously Awesome

April Dunford's structure is the core model for this skill:

- Competitive alternatives
- Unique attributes
- Value those attributes unlock
- Best-fit customers
- Market category

The sequence matters. Start from alternatives and best-fit customers, then derive the rest.

## Positioning guardrails

- Include status quo when it is a real alternative.
- Write differentiated value as an outcome, not as a feature list.
- Use category framing to help understanding, not to overreach.
- Add explicit disqualification language when the product only fits a narrow buyer or motion.
