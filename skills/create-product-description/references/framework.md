# Product Description Framework

Always include:

1. Product positioning
2. Description lengths
3. Full product description structure
4. SEO optimization
5. Emotional triggers
6. Trust elements
7. Comparison copy
8. Channel adaptations
9. Seasonal or promotional variants
10. Testing recommendations

## Guardrails

- No opening commentary.
- Benefits before features.
- Keep the output scannable.
- Integrate SEO naturally.
