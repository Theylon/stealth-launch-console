# Referral Program Framework

Always include:

1. Program foundation analysis
2. Customer motivation analysis
3. Incentive structure design
4. Viral loop design
5. Channel strategy
6. Program mechanics
7. User experience flow
8. Messaging framework
9. Implementation roadmap

## Guardrails

- No opening commentary.
- Tie incentives to LTV, CAC, and margin logic.
- Make referral rules and anti-fraud measures explicit.
- Keep the system easy for customers to understand.
