# Create GTM Strategy

## Overview

Create a practical GTM plan for launch or market entry, covering segments, channels, pricing, assets, enablement, metrics, and risks. Use this when the user needs an execution-oriented plan rather than only messaging or positioning.

Read [references/framework.md](references/framework.md) for the required sections and phase structure.

## Workflow

1. Confirm the launch object, audience, timeline, and budget.
2. Define the opportunity, priority segments, and anti-personas.
3. Lock the positioning and value pillars that the GTM will carry.
4. Build channel mix, phases, content plan, enablement, and metrics.
5. Finish with budget allocation and risk mitigation.

## Output Contract

- Start directly with the GTM framework.
- Keep timelines realistic.
- Make metrics, targets, and ownership practical for the stated stage.
- Include only tactics that fit the company size and motion.

## Quality Bar

- Favor specificity over exhaustive channel lists.
- Make the plan stage-appropriate.
- Tie channel and asset recommendations back to the audience and launch goal.
