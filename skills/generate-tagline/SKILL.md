# Generate Tagline

## Overview

Create a large set of tagline options, score them, and narrow to the strongest finalists with practical usage guidance. Use this when the user needs naming-like rigor for a brand line rather than a few casual slogan ideas.

Read [references/framework.md](references/framework.md) for the option categories and evaluation model.

## Workflow

1. Distill the brand essence, differentiator, and emotional promise.
2. Generate options across the required tagline categories.
3. Score the strongest options for memorability, clarity, differentiation, and versatility.
4. Recommend the top finalists with rationale and usage examples.
5. Add testing, rollout, and trademark considerations.

## Output Contract

- Start directly with the framework output.
- Provide broad variety across categories before converging.
- Include rationale, not just raw options.
- Keep implementation notes practical rather than legalistic.

## Quality Bar

- Avoid generic category cliches.
- Finalists should work in real brand contexts, not just as clever phrases.
- Distinguish between master brand lines and campaign lines.
