# Positioning Canvas Framework

Always include:

1. Market category and frame of reference
2. Market statistics summary with sources
3. Competitive analysis table
4. Unique attributes matrix
5. Value proposition and proof
6. Customer segments analysis
7. Positioning statement framework
8. Messaging hierarchy
9. SWOT analysis
10. Strategic recommendations

## Guardrails

- No intro or outro text.
- Cite real sources for statistics and evidence.
- Support claims with proof.
- Do not invent market data.
