# Create Positioning

## Overview

Create a detailed positioning canvas when the user needs category research, competitive framing, evidence-backed differentiation, and strategic recommendations in one artifact. This skill is broader and more research-heavy than `$position-product`.

Read [references/framework.md](references/framework.md) for the required sections and evidence rules.

## Workflow

1. Confirm the company, product, audience, and any known competitors.
2. Research the market category and competitor set when the user has not supplied enough context.
3. Gather only credible, verifiable market statistics and cite sources.
4. Build the positioning canvas from category frame through strategic recommendations.
5. Keep proof, statistics, and claims sourced or explicitly marked as assumptions.

## Output Contract

- Output structured markdown only.
- Start directly with the first section.
- Include citations or links for real market data and proof claims.
- Do not fabricate TAM, SAM, SOM, competitor claims, or proof points.

## Quality Bar

- Use only credible, verifiable sources.
- Distinguish direct competitors, indirect alternatives, and DIY/status quo.
- Focus on unique attributes that materially matter to the buyer.
