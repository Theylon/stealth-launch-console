# FAQ Framework

Always include:

1. Question categories
2. Complete FAQ set
3. Objection-handling FAQs
4. SEO-optimized FAQs
5. Schema markup ready output
6. Tone and voice guidelines
7. FAQ page structure
8. Maintenance plan
9. Channel adaptations
10. Measurement

## Guardrails

- No opening commentary.
- Answer directly first.
- Use natural question phrasing.
- Include actionable next steps and useful cross-links where relevant.
