# Category Positioning Framework

Always include:

1. Current category analysis
2. Category opportunity assessment
3. New category design
4. Category point of view
5. Category differentiation matrix
6. Lightning strike strategy
7. Category ecosystem
8. Messaging architecture
9. Competitive response playbook

## Guardrails

- No opening commentary.
- Be differentiated, but keep the rollout realistic.
- Recommend one primary category move with rationale.
