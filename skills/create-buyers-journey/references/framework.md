# Buyers Journey Framework

Always include:

1. Journey stage overview
2. Detailed stage analysis for:
   - Unaware
   - Problem aware
   - Solution aware
   - Product aware
   - Most aware
3. Touchpoint mapping
4. Content needs matrix
5. Objection timeline
6. Conversion triggers
7. Friction points and solutions
8. Measurement framework

## Guardrails

- No opening commentary.
- Keep the journey company-specific.
- Make content and measurement recommendations actionable.
