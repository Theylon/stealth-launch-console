# Generate Persona Framework

Use this skill to create one detailed persona with the following sections:

1. Professional profile
2. Categorized motivations
3. Categorized pain points
4. Information consumption patterns
5. Content marketing strategy
6. Event marketing strategy
7. Decision journey mapping
8. Marketing engagement guidelines
9. Content calendar planning
10. Messaging frameworks

## Top-level JSON shape

Use these exact keys:

- `Name`
- `MarketType`
- `RoleJob`
- `CompanySize`
- `Demographics`
- `ProfessionalProfile`
- `CategorizedMotivations`
- `Goals`
- `Aspirations`
- `CategorizedPainPoints`
- `InformationConsumptionPatterns`
- `ContentMarketingStrategy`
- `EventMarketingStrategy`
- `DecisionJourneyMapping`
- `MarketingEngagementGuidelines`
- `ContentCalendarPlanning`
- `MessagingFrameworks`
- `VoiceOfCustomer`
- `PrimaryRisk`
- `StrategicOpportunity`

## Guardrails

- Generate one persona only.
- Infer B2B vs B2C from the context.
- Use realistic KPIs, budgets, and reporting lines.
- Keep every field specific and usable.
