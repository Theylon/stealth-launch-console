# Landing Page Framework

Always include:

- Hero section
- Problem and agitation
- Solution introduction
- How it works
- Social proof stack
- Benefits breakdown
- Features showcase
- Why choose us
- Objection handling
- Urgency and scarcity
- Authority and credibility
- Value stacking
- Risk reversal
- Final push
- Footer trust elements

## Guardrails

- No intro or outro commentary.
- Use specific copy for each section.
- Keep the structure conversion-oriented and persona-specific.
- Apply persuasion ethically.
