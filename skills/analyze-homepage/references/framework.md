# Homepage Analysis Framework

Always include:

1. LIFT model scoring
2. First impression audit
3. Value proposition analysis
4. Navigation and information architecture
5. CTA analysis
6. Trust signals audit
7. Content hierarchy analysis
8. Messaging clarity
9. Visual design assessment
10. Technical performance
11. Competitive comparison
12. Prioritized recommendations
13. Form friction analysis if relevant
14. Micro-conversion analysis
15. A/B test roadmap

## Guardrails

- No opening commentary.
- Include the total LIFT score.
- Recommend specific before-or-after style fixes where possible.
- Prioritize by likely impact.
