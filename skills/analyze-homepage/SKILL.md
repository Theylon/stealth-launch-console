# Analyze Homepage

## Overview

Audit a homepage for conversion performance using a structured CRO lens. Use this when the user needs broader homepage analysis than copy quality alone, including UX, trust, IA, and performance considerations.

Read [references/framework.md](references/framework.md) for the LIFT model structure and audit sections.

Use external benchmarks when you need to separate local opinion from broader pattern evidence:

- Use [Refero](https://refero.design/) to compare the page against current homepage and landing-page conventions, section patterns, and UX flows.
- Use [component.gallery](https://component.gallery/) to evaluate whether specific components or interaction patterns are strong, weak, outdated, or mismatched for the job.
- Do not treat either site as authority by itself. Use them to sharpen diagnosis, not to justify shallow trend-chasing.

## Workflow

1. Confirm the homepage target audience and primary conversion goal.
2. Identify the page archetype and gather a small benchmark set from Refero when the comparison class matters.
3. Score the page against the LIFT model.
4. Review above-the-fold clarity, navigation, CTA hierarchy, trust, messaging, visual design, and component quality.
5. Use component.gallery when a recommendation depends on component-level behavior such as nav, tabs, FAQ, pricing, forms, tables, or cards.
6. Add performance, SEO, form-friction, and micro-conversion analysis where relevant.
7. End with a prioritized optimization roadmap and A/B tests.

## Output Contract

- Start directly with the audit output.
- Include a visible LIFT score.
- Prioritize findings by impact.
- Tie every recommendation to a concrete issue on the page.
- When benchmark research was used, distinguish page-level benchmark gaps from component-level issues.

## Quality Bar

- Use specific, actionable recommendations.
- Separate quick wins from structural redesign work.
- Do not confuse messaging fixes with UX or funnel issues.
- Do not recommend changes only because they are common elsewhere; explain why they improve this page for this buyer and conversion goal.
