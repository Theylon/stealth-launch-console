# Value Proposition Ladder Framework

Always include:

- Feature -> Capability -> Benefit -> Outcome mapping table
- 5 to 7 complete ladders
- Role-based value propositions for executive, manager, and user audiences
- Audience-specific messaging matrix for 3 segments
- Website messaging guide
- Proof point strategy

## Guardrails

- No opening commentary.
- Use customer language.
- Tie claims to evidence wherever possible.
- If proof is absent, call out the missing evidence explicitly.
