# Create Press Release

## Overview

Write a newsworthy press release in AP-style structure and package it with the supporting materials needed for distribution. Use this when the user needs both the release and the launch plan around it.

Read [references/framework.md](references/framework.md) for the full structure and output variants.

## Workflow

1. Confirm the announcement type, key details, spokesperson, and available proof.
2. Assess the strongest news angle and lead with it.
3. Write the press release in inverted-pyramid order.
4. Add quote options, key messages, multimedia guidance, and distribution strategy.
5. Provide short, medium, and long variants where the framework calls for them.

## Output Contract

- Start directly with the framework output.
- Use AP-style press release formatting.
- Ensure quotes add perspective rather than repeat the lead.
- Make the distribution plan specific to the announcement.

## Quality Bar

- The angle must feel genuinely newsworthy.
- Proof points should strengthen the release, not overload it.
- Avoid hype that would weaken media credibility.
