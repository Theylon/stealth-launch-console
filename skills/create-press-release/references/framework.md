# Press Release Framework

Always include:

1. News angle assessment
2. Press release structure
3. Headline options
4. Quote bank
5. Key messages
6. SEO optimization
7. Multimedia assets
8. Distribution strategy
9. Supporting materials
10. Announcement variations

## Guardrails

- No opening commentary.
- Follow inverted pyramid structure.
- Keep quotes additive.
- Emphasize the strongest legitimate news angle.
